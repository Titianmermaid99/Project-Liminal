# Project Summary

Project Liminal is a cooperative virtual reality psychological horror exploration game designed for standalone Meta Quest headsets.

Players work together to navigate procedurally generated liminal environments, manage their survival, uncover environmental stories, and evade hostile entities whose behaviour challenges perception and trust.

Rather than relying on frequent jump scares, Project Liminal focuses on atmosphere, uncertainty, and psychological tension.

Every mechanic is designed to reinforce player immersion.

Current Sprint

A1 – Core XR Foundation

Current Focus

Establishing the project's core XR framework for controller-based virtual reality gameplay.

Sprint A1 will create the stable technical foundation required for future interaction, locomotion, multiplayer, procedural generation, and gameplay systems while maintaining Meta Quest 2 performance as the minimum target.

# Success Criteria

Project Liminal will be considered successful when it delivers:

- A highly immersive VR experience.
- Strong cooperative gameplay.
- Psychological horror driven by atmosphere rather than cheap scares.
- Stable standalone VR performance.
- High replayability through procedural generation.
